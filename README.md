## Build deps
```
CMake 3.14 or greater
GCC (part of MinGW for Windows)
Make (part of MinGW for Windows)
```
## Build instructions
```
mkdir build
cd build
cmake -G "MinGW Makefiles" ..
make
```
Binaries will be in the build/bin
